Для установки необходимых модулей впишите в консоль из директории c основным файлом main.py 'python3 setup.py -i'

Для работы с приложением запустите main.py: 'python3 main.py'

Базы данных собранных с аккаунта социальной сети пользователей хранятся в папке соответствующей социальной сети.

Для публикации изображений или видео убедитесь, что изображения и видео хранятся в папке resources.
